To open cmd terminal:
Ctrl + Alt + t

Current directory path:
pdw

One step backword directory:
cd ..

Folder and file list:
ls

Folder and file list (with details):
ls -l

Details formate is: 
drwxr-xr-x arif arif f/d_size modified_time f/d_name
file or directory, access for user, group, others username, groupname, f/d_size, time, f/d_name

Vsiting a specific directory:
cd folder_name
or, cd /file_path

Displaing file content:
cat filename
or, cat /file_path

Creating a folder:
mkdir foldername

File create:
touch filename

Coping file:
cp source_path destination_path

Coping directory
cp -r source_path destination_path

Rename file/directory
mv current_name new_name

Delete file
rm source_path
or, rm -f source_path

Delete directory
rm -r source_path
or, rm -rf source_path

Permission change file:
chmod
*if chmod not work, use 'sudo chmod'

Permission change directory
chmod -R 756 dir_name
756 stands for: rwx r-x rw-
*if chmod not work, use 'sudo chmod'
